# MPI Async Overlap Study

This project explores the asynchronous overlap capabilities of the MPI (Message Passing Interface) standard. The focus is on analyzing the performance implications and visualization results of various overlap scenarios.

## Visualization Results

The visualization results can be found in the following folders:
- `visualization/results1`: Contains results for scenario 1.
- `visualization/results2`: Includes outputs from scenario 2.
- `visualization/results3`: Displays the results for scenario 3.
